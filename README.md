## Install
Download the flex_box_plus.css and import it into your project. Then add the following to your index.html file
```
<link rel="stylesheet" href="flex_box_plus.css">
```

## Example
```
<div class="x-center-center">
    <p>item 1</p>
    <p>item 2</p>
    <p>item 3</p>
</div>

<div class="y-apart-end">
    <p>item 1</p>
    <p>item 2</p>
    <p>item 3</p>
</div>
```

## arguments
```
<axis>-<main>-<cross>

(axis)
x - horizontally
y - vertically

(main)
start
center
end
even
apart

(cross)
start
center
end
stretch
```
